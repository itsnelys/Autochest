## Français

### Présentation
Autochest est un script Space Engineers de tri automatique d'inventaire. Il repartit les ressources entre stockage principal, glace, minerais et inventaires proteges. Il sert a maintenir une base ou un vaisseau organise sans transfert manuel constant.

### Fonctionnalités
- Tri automatique des ressources et objets.
- Protection des inventaires critiques comme reacteurs, generateurs, armes et machines.
- Detection des conteneurs de destination par nom ou groupe.
- Journalisation et etat sur LCD.

### Installation
Ajoute Script_Autochest.cs dans un Programmable Block Space Engineers, puis nomme ou groupe les conteneurs selon la configuration du script.

### Utilisation
Lance le script depuis le Programmable Block apres avoir cree les conteneurs de destination. Utilise la commande de scan si tu modifies les blocs ou les groupes.

## English

### Overview
Autochest is a Space Engineers script for automatic inventory sorting. It routes resources between main storage, ice, raw resources and protected inventories. It keeps a base or ship organized without constant manual transfers.

### Features
- Automatic sorting for resources and items.
- Protection for critical inventories such as reactors, generators, weapons and production blocks.
- Destination container detection by name or group.
- LCD status and transfer logging.

### Installation
Add Script_Autochest.cs to a Space Engineers Programmable Block, then name or group containers according to the script configuration.

### Usage
Run the script from the Programmable Block after creating destination containers. Use the scan command when blocks or groups change.
